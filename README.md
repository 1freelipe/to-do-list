<div align='center'>
<h2>TO DO LIST</h2>

<p>Sistema de gerenciamento de tarefas desenvolvido para facilitar a organização pessoal por categorias.
Permite cadastro e autenticação de usuários, criação e edição de tarefas, filtragem por categorias e visualização de tarefas finalizadas.</p>
</div>

<h3>Recursos Principais</h3>
<nav> 
<ul>
<li>Cadastro de usuário</li>
<li>Login seguro</li>
<li>Organização de Tarefas</li>
<li>Adicionar tarefas</li>
<li>Criar categorias personalizadas</li>
<li>Filtrar tarefas por categoria</li>
<li>Editar tarefas existentes</li>
<li>Marcar como concluída</li>
<li>Visualizar tarefas finalizadas</li>
</ul>
</nav>
